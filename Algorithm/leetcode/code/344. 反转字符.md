[344.反转字符](https://leetcode-cn.com/problems/reverse-string/)
***
这个for循环的亚子，我爱了；  
面试也许不能用包函数；

48ms
```cpp
class Solution {
public:
    void reverseString(vector<char>& s) {
        ios::sync_with_stdio(false); 
        int i,j;
        for(i=0,j=s.size()-1;i<j;i++,j--) swap(s[i],s[j]);
    }
};
```

```cpp
class Solution {
public:
    void reverseString(vector<char>& s) {
        int i=0,j=s.size()-1;
        char tmpc;
        while(i<j){
            tmpc=s[i];
            s[i]=s[j];
            s[j]=tmpc;
            i++,j--;
        }
    }
};
```
